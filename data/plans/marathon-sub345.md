# Marathon Training Plan — Sub 3:45

## Overview

- **Target event:** Marathon
- **Goal time:** Sub 3:45 (5:20/km race pace)
- **Plan duration:** 16 weeks

## Pace Zones

| Zone | Pace | Description |
|------|------|-------------|
| Easy | 6:00/km | Recovery and aerobic base |
| Steady | 5:19/km | Comfortable-hard; aerobic threshold |
| Tempo | 5:06/km | Sustained threshold effort |
| Fast | Variable | Intervals; effort-based (hard) |
| Race | 5:20/km | Target marathon pace |

## Phase Structure

| Phase | Weeks | Focus |
|-------|-------|-------|
| Base | 1–4 | Build volume, introduce intervals and tempo, 8–10k race |
| Build | 5–9 | Longer long runs, progressive tempo/steady, half marathon race |
| Peak | 10–13 | Highest volume, longest long runs (32–33.6 km), race-specific |
| Taper | 14–16 | Volume reduction, maintain sharpness, race day |
